window(브라우저) object(객체)

브라우저 객체는 사용자가 사용하는 웹브라우저의 창에서 다양한 동작이나 속성을 설정하고자 할 때 사용하는 객체이다.

window 객체

window 객체는 브라우저 객체중 최상위 객체로서 현재 실행되고 있는 브라우저창을 나타낸다.

window 객체의 속성(property)
document
frameElement
innerHeight
innerWidth
location
name
screenX, screenY
scrollX, scrollY
sessionStorge
Window 객체의 메서드(method)
alert() - 윈도우창에서 경고창(메세지)를 띄울 때 사용
blur() - 현재창에서 포커스를 제거
close() - 현재창을 닫는다
confirm() - 확인, 취소 버튼이 있는 확인창 표시
focus() - 현재 창에 포커스를 부여
moveBy() - 현재 창을 지정한 크기만큼 이동
moveTo() - 현재 창을 지정한 좌표로 이동
open() - 새로운 창을 엶
prompt() - 사용자가 입력가능한 프롬프트창을 엶
resizeTo() - 브라우저 창 크기를 조절할 때 사용
scroll() - 문서의 특정 위치로 스크롤
sizeToContent() - 내용에 맞게 창의 크기 조절
stop() - 로딩중지

팝업창은 브라우저의 팝업차단 기능으로 인해 사용자가 보지 않을 가능성이 커 요즘 팝업창을 띄우지 않고 팝업레이어, 모달윈도(css로 박스 포지션 배치)를 활용하여 띄운다.

모달 윈도(modal window)
모달 윈도는 사용자 인터페이스 디자인 개념에서 자식 윈도에서 부모 윈도우로 돌아가기 전에 사용자의 상호동작을 요구하는 창을 말한다.
모달 윈도우는 부모창을 배경으로 어둡게 하고 자식요소가 화면 중앙에 배치하도록 하며 오른쪽 상단에 '닫기'버튼을 만들어서 닫을 수 있도록 한다.
