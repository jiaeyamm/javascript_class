01_ 자바스크립트 소개와 기초문법 p.441

자바스크립트는 html 문서 안에서 삽입되어 동작된다.

기본적으로 CSS 언어와 같이 head 안에서 작성되며, 필요에 따라 body 태그 안에서 작성이 가능하다.

자바스크립트는 주석을 작성할 때 2가지 종류가 있으며 한줄 주석과 두줄이상 주석이 있다. 한줄 주석은 //
두줄 이상 주석은 /**/

html, CSS언어는 대,소문자 구별 안 하지만 자바스크립트는 대,소문자를 구별한다.

자바스크립트에서는 여러 데이터 형식이 존재하는데 그 중에서 '문자데이터'를 표시할 때 '', "" 구분기호를 사용해야 한다.

자바스크립트는 CSS처럼 현재 문서에서만 적용할 수 있는 '내부 스크립트'와 별도의 js 문서를 작성하여 html 문서에 연결해서 사용하는 '외부 스크립트'가 있다.

스크립트 작성 시 script, /script 로 종료한다.
