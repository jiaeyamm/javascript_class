제이쿼리_01_animate메서드

제이쿼리에서는 효과와 애니메이트 메소드를 사용하여 사용자가 선택한 요소를 다양하게 움직이도록 할 수 있다.

1. animate 메소드 작성법

$('대상').animate( { '속성' : '값' }, 시간, 가속도, 콜백함수 );

{ '속성' : '값' } - css 속성과 값을 말하며, '값'에는 자바스크립트 변수나 연산자를 사용할 수 있다.

시간 - 애니메이션이 동작하는 시간을 말하며 1/1000초 = 1초를 나타냄

가속도 - easing메소드를 활용한 값을 말함 (이미지 참조)

콜백함수 - callback function 으로 애니메이션 동작이 끝나면 다음 동작을 이어 실행하고자 할 때 사용.
