내장객체_배열(array)

배열(array)객체란?
-연관된 데이터를 하나처럼 관리하고 싶을 때 사용
-복수의 데이터를 제어할 수 있는 다양한 함수를 제공
-배열의 함수를 이용한 데이터를 편리한 조작이 가능

연관된 데이터 정보 - 과일정보, 회원정보, 쇼핑정보, 예약정보, 호텔예약정보....

1. 배열 객체 선언하기

방법 1. let array - new Array(data1, data2, data3, datea4.... data N)

방법 2. let array = [data1, data2, data3, data4.... dataN]

예 ) let food = new Array ('사과', '감', '오렌지', '귤', '키워', '포도' ...)

2. 배열 객체 호출방법 - array[index번호]

배열 객체를 호출할 때에는 index 번호를 사용하는데 index 번호는 0부터 시작한다.

배열 객체 메서드

concat - 기존 배열에 다른 배열을 추가

every - 배열에 대한 모든 요소가 주어진 함수에 true를 반환 아니면 false 반환

filter - 배열 요소 중에서 주어진 함수에 대해 true인 것만 반환

forEach - 배열의 모든 요소에 대해 주어진 함수를 실행

indexOf - 주어진 값과 일치하는 값이 있는 배열 요소의 첫 인덱스를 찾아줌
