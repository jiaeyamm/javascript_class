01_jqeury DOM_traversing(문서객체탐색)

문서 객체 모델(DOM) - 제이쿼리 traversing(탐색) 메서드는 기존의 선택자보다 좀더html문서 요소들을 정확하게 선택해주는 메소드이다.

CSS 선택자 부모아래 자식 요소 선택은 .부모 > 자손, '부모 자손', 형제 요소 선택시 'a + b', 'a ~ b' 방법을 사용한다.

제이쿼리에서는 html 문서와 구조에서 부모요소를 찾거나 부모의 형제들, 부모아래 자식요소들, 자식요소에서 부모의 부모들을 찾을 수 있는 메소드들이다.

1. 부모요소 찾기

  • parent(); - 선택한 요소의 바로 위에 위치한 부모요소를 선택한다.
  • parents(); - 선택한 요소의 위에 위치한 모든 부모요소를 선택한다.
  • parentUntil(); - 선택한 요소에서부터 지정한 요소전까지 부모요소를 선택한다.

2. 자식요소 찾기

  • children();
  • find();

3. 형제요소(sibling) 찾기

  • siblings(); - 현재 요소의 형재관계인 요소를 모두 선택한다.
  • next(); - 현재 요소의 다음애 오는 요소를 선택
  • nextUntil() - 현재 요소의 다음에서부터 ~ 이전까지를 선택
  • nextAll() - 현재 요소의 다음에 있는 모든 요소를 선택
  • prev() - 현재요소의 이전에 있는 요소를 선택
  • prevAll() - 현재요소의 이전에 있는 모든 요소를 선택
  • prevUntil() - 현재 요소의 이전에 있는 요소부터 ~전까지를 선택

4. 탐색(traversing) 메서드 - filtering

html문서 구조에서 사용자가 선택한 요소를 정확하게 찾아낼 수 있는 메소드 중의 하나이다.

  • first() - 형재관계 요소들 중에 첫번째 요소 선택
  • last() - 형재관계 요소들 중 마지막 요소 선택
  • eq(n) - 인덱스값의 n번째에 해당하는 요소를 선택
  • filter() - 사용자가 지정한 이름에 맞는 요소를 선택
  • not() - 사용자가 지정한 이름이 아닌 요소를 선택
